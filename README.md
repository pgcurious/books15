### ðŸ”„ Analogical Thinking
Complex AI concepts become intuitive through carefully chosen analogies. An LLM is like a brilliant colleague; tools are like giving that colleague hands; memory is like giving them a notebook.

## Prerequisites

- Basic Python knowledge
- Understanding of APIs (helpful but not required)
- Curiosity about AI and its capabilities

## Setup

```bash
# Clone the repository
git clone <repository-url>
cd agentic-ai-foundations

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

## Environment Variables

Create a `.env` file in the root directory:

```
OPENAI_API_KEY=your_openai_api_key_here
LANGCHAIN_API_KEY=your_langchain_api_key_here  # Optional, for tracing
```
